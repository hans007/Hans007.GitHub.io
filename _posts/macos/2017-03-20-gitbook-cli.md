# 前言

## 安装

```
npm install -g gitbook-cli  
```

> 需要nodejs环境

## 命令 - 本地编译

```
gitbook build
```

## 命令 - 本地WEB服务方式

```
gitbook serve
```

## 命令 - 初始化

```
gitbook init
```

> 这将会生成一个标准项目

## `SUMMARY.md` 文件

```
# Summary

* [Introduction](README.md)
```

> 这里存放了目录与文件的对应关系
